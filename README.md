# presentation-why-es6

## How start create presentation like this...

```
echo '{}' > package.json
echo '/node_modules/' > .gitignore
npm i --save shower
cp node_modules/shower/index.html .
```

Next step is fixed path to main `*.css` and `*.js` files (add `node_modules/shower/`).
